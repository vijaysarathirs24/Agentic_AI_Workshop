# 🎯 Placement Match Maximizer

**Placement Match Maximizer** is a smart AI-powered Streamlit application designed to help students and freshers align their resumes, projects, or GitHub profiles with top tech job roles in 2025. It uses **Gemini 2.0 Flash** and **Tavily Search** to analyze user input and generate customized job role matches, rewritten resume points, company suggestions, LinkedIn posts, and placement tips.

---

## 🚀 Features

- 🔍 **Analyze Resumes (PDF/TXT), GitHub Repos, or Project Writeups**
- 💼 **Match with Top 2025 Tech Job Roles**
- ✍️ **Rewrite Resume/Project Bullet Points for Better Alignment**
- 🏢 **Suggest Companies Hiring for Relevant Roles**
- 💡 **Short LinkedIn Post Generator**
- 📚 **Course & Certification Recommendations**
- 💬 **Interview and Placement Tips**

---

## 🛠️ Tech Stack

- [Streamlit](https://streamlit.io/) – For UI
- [LangChain](https://www.langchain.com/) – For prompt orchestration
- [Gemini 2.0 Flash (Google Generative AI)](https://ai.google.dev/) – For LLM-based generation
- [Tavily Search API](https://www.tavily.com/) – For real-time recruiter feedback and job role discovery
- [pdfplumber](https://github.com/jsvine/pdfplumber) – For PDF resume parsing
- [GitHub API](https://docs.github.com/en/rest) – For fetching README from public repositories
